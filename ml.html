What is Machine Learning?

Machine learning (ML) is a subset of Artificial Intelligence (AI) that consists of algorithms that reflect how humans learn, hence teaching technology how to learn. Initially, when these programs are written, they are very inaccurate and have many flaws, but over a prolonged time, the technology becomes more accurate. (Hosch, 2024)

  1. Decision
  2. Error
  3. Optimisation

The machine must be able to identify the error made and adjust and find an alternative solution to the problem. By repeating this process, the technology becomes faster and more effective over time, becoming an invaluable tool for businesses. (Stryker & Kavlakoglu, 2024)

What are the typical uses for Machine Learning?

ML has many uses in modern-day technology, some more commonly known than others. A primary example is the ever-improving biometric systems. Many Apple and Android devices use biometric scanners, such as Touch ID and Face ID, which were added to Apple Devices towards the end of 2017. (Cipriani, 2020)

Biometric systems use ML to observe patterns within the biometric scan. When Touch ID is initially set up on devices, the biometric system scans for unique patterns in the fingerprint and uses algorithms to register only as correct when that specific pattern is detected. (Reddy et al., 2023)

Another common example is product recommendations on retail websites, such as Amazon and online retailers. It uses ML to observe the users purchases, clicks and search history. The algorithm then filters the information to recommend products based on what it has observed about the user. Many different algorithms are used for this, but Collaborative Filtering is most commonly used. (NVIDIA, 2021)
