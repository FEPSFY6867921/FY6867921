Abstract

Globally, company investments have skyrocketed such that “91% of top businesses have ongoing investments” (O’Donnellan, 2024) in AI and Machine Learning (ML). Despite its prevalence in modern-day society, few understand its true capabilities, which is why I have investigated the effectiveness of ML in Cybersecurity and how it is used to defend against Data Mining. ML is a subset of AI and is an algorithm that reflects how humans learn (Hosch, 2024) by problem identification, error identification and optimisation (Stryker & Kavlakoglu, 2024). ML has many different uses, the most common being in Cybersecurity, which is the process of defending against cyber-attacks through ML using threat detection algorithms to respond to security incidents” (IBM, 2024). Data Mining is a set of algorithms that analyse data for patterns and links. Organisations are at risk of cybercriminals using Data Mining to quickly search for data to steal (Smith, 2009), where ML uses “highly predictive models” (Shams, 2023) to identify targets and protect them accordingly.
